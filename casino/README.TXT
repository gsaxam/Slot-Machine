PROJECT TITLE: Casino Stout

PURPOSE OF PROJECT: Create a casino style Slot Machine game that uses card game symbols.

VERSION or DATE: 12/15/2010

HOW TO START THIS PROJECT AND USER INSTRUCTIONS:
         The project has a main world class and an actor class. The main world class is the 
casino class. The main Actors are spinButton, heart, diamond, spade, club, ScoreBoard, help ,
 playagain and betLabel.Other actors are created for various information passing and displaying
 purposes. The casino class has all the definedobjects that are visible during execution. All
 instances of major class that contribute to the game execution are listed in this class. All
 other actor class have their specific functions. The actor class spinButton contains methods
 form playing an animation, spin(), checkWinner(), clearScreen(), stopAllSounds(), hasEnough()
 methods.
